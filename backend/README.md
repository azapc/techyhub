# Backend - TechStore API

REST API built with **NestJS 11** + **TypeScript** + **Prisma 7** + **PostgreSQL 16**.

- **Port:** 3001
- **Swagger:** [http://localhost:3001/api/docs](http://localhost:3001/api/docs)

## Quick Start

```bash
pnpm install
pnpm start:dev
```
