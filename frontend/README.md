# Frontend - TechStore Admin

Admin dashboard built with **Nuxt 4** + **Nuxt UI v4** + **Pinia** + **Tailwind CSS 4**.

- **Port:** 3000
- **Admin:** [http://localhost:3000/admin](http://localhost:3000/admin)

## Quick Start

```bash
pnpm install
pnpm dev
```
